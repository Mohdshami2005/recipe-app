<h1 align="center">Recipe App</h1>

<p align="center">
A responsive Recipe Search web application built using
<b>HTML</b>, <b>CSS</b>, and <b>JavaScript</b>.
Search for any recipe, view its ingredients and cooking instructions,
or watch its YouTube recipe video.
</p>

<h2>📌 Features</h2>

<ul>
    <li>Search recipes by name.</li>
    <li>Display recipe image, name, category, and cuisine.</li>
    <li>View complete ingredients and cooking instructions.</li>
    <li>Open the recipe's YouTube tutorial in a new tab.</li>
    <li>Fully responsive design for desktop and mobile devices.</li>
    <li>Fast recipe fetching using the Fetch API.</li>
    <li>Proper error handling for invalid searches.</li>
</ul>
<h2>🛠️ Built With</h2>

<ul>
    <li>HTML5</li>
    <li>CSS3</li>
    <li>JavaScript (ES6)</li>
    <li>Fetch API</li>
    <li>CSS Grid & Flexbox</li>
    <li>Font Awesome</li>
</ul>

<p>
This project uses the free
<b>TheMealDB API</b>
to fetch recipe information including:
</p>

<ul>
    <li>Recipe Name</li>
    <li>Recipe Image</li>
    <li>Category</li>
    <li>Cuisine</li>
    <li>Ingredients</li>
    <li>Measurements</li>
    <li>Cooking Instructions</li>
    <li>YouTube Tutorial Link</li>
</ul>

<h2>⚙️ How It Works</h2>

<ol>
    <li>Enter the name of a recipe in the search box.</li>
    <li>Click the <b>Search</b> button.</li>
    <li>The application sends a request to <b>TheMealDB API</b> using the Fetch API.</li>
    <li>Matching recipes are displayed as responsive recipe cards.</li>
    <li>Click <b>View Recipe</b> to see the ingredients and cooking instructions in a popup.</li>
    <li>Click anywhere on a recipe card (except the button) to watch its YouTube tutorial in a new browser tab.</li>
</ol>
